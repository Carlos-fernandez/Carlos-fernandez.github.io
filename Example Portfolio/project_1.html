Pipette Gravimetric Fixture
Aim:

The purpose of this project is to developed a fixture that can validate single-channel and eight channel pipettes for precision and accuracy. Ultizing the will ensure reliable results that a customer would obtain for pipette calibration.

Approach:
  • Protect analytical balance from air flow with acrylic housing.
  • Eliminate vibration forces from robot by using separate tables.
  • Design a test script that automates the motion control, communication to to analytical balance], and extract a clean data collection.
  • Contruct a protocol module for the scale to extract readings
  • PASS/FAIL Criterion
  • Ease of use for operators
