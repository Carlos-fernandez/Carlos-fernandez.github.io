Pipette Gravimetric Fixture
Brief

The purpose of this project is to developed a fixture that can validate single-channel and eight channel pipettes for precision and accuracy. Ultizing a custom robot with a modified deck will ensure reliable results that a customer would obtain for a pipettes liquid handling capablies.

Development:
  • Protect analytical balance from air flow with acrylic housing.
  • Eliminate vibration forces from robot by using separate tables.
  • Design a test script that automates the motion control, communication to to analytical balance], and extract a clean data collection.
  • Contruct a protocol module for the scale to extract readings.
  • PASS/FAIL Criterion.
  • Ease of use for operators.
  • Provide a certificate for pipette calibration for customer use.

Structure of Algorithm

  • QC pipette script handles the main execution of the actual test, that includes function test, volume test qc, and multi-dispense test for any pipette.
  • The calibration script allows the operator to save positions for each labware on the deck. This is required to run before the QC pipette script.
  • All pipettes function with the test script
  • If a new pipette is introduce, a production staff member will need to add a new version of the pipette to the labware_config.json and the test_config.json

    • Three Python Libraries

  • Data process handles all the csv processing for each pipette and to load the configuration json files for the test.
  • Rig Function library is predefined functions that handle the test functionality.
  • Robot Functions library handles labware calibration functions and basic robot functions. For example: Move to Location, pick up tip, drop tip, trash tip, etc.
