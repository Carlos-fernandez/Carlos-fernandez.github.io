Makerbot Replicator 5+ Gantry Tester
Brief:

The purpose of the Gantry tester is to assess the parallelism of the two linear rails on the X Stage of the Replicator 5+ by evaluating the frictional force within the mechanical system. The test was specifically designed to distinguish between properly assembled and poorly assembled gantries. The primary goal of this fixture is to guarantee that customers receive a well-assembled top bunk mechanical system, ultimately ensuring excellent print quality. Furthermore, alongside developing the fixture, I also created a comprehensive test script that provides a PASS/FAIL criterion for efficient evaluation.

Development:
  • Design a fixture that automates the data collection and outputs into a csv file.
  • Create a test script that prompts the user with instructions.
  • Create a PASS/FAIL Creterion.
  • Write an standard operation procedure(SOP) for production.
  • Develop a fixture that automates data collection and exports the data into a CSV file.
  • Design a user-friendly test script that provides clear instructions to guide the user through the testing process.
  • Establish a reliable PASS/FAIL criterion to evaluate the test results effectively.
  • Prepare a comprehensive Standard Operating Procedure (SOP) specifically tailored for production purposes, ensuring standardized procedures are followed consistently.
